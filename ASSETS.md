# Assets

**Art direction:** Dark fantasy 2D card game with clean flat-color digital art, bold outlines, muted earth-tone dungeon palette (browns, greys, dark reds) with vibrant card accents (red for attacks, green for skills). STS-inspired UI layout — character sprites on left, enemies on right, card hand fanned at bottom, HP bars above entities, energy orbs bottom-left. Semi-transparent UI panels blending with the stone dungeon background. Characters have bold silhouettes readable at game resolution.
